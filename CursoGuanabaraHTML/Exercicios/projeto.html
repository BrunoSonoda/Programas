O Mundo da Tecnologia

(Um Cordel para os Programadores)

No tempo da programação,
O código é poesia,
Quem digita com paixão
Cria um mundo de magia.

O Java já se consagra,
Na tela faz seu papel,
Quem domina essa palavra
Constrói até um castelo fiel.

Eclipse brilha na tela,
Com classes a se juntar,
Quem conhece essa novela
Sabe bem como compilar.

O código bem escrito
Vale mais que um diamante,
Se erra um ponto ou um dito,
O bug surge num instante.

Na lógica e na razão,
O sistema ganha vida,
Mas sem uma boa função,
A máquina fica perdida.

No banco de dados guardado,
Está tudo organizado,
Mas se a chave se perder,
Dá trabalho pra valer!

Se a API dá resposta,
O dev fica contente,
Mas se a falha encosta,
O erro chega na frente!

E assim segue o programador,
Com café e muita garra,
Transformando o mundo em cor
Com um código que não para!
